<feature>
  <name>Context-batching and multi-engine dispatch</name>
  <files>tests/test_translator.py</files>
  <behavior>
**Batching tests (Argos engine):**
- `test_translate_segments_batches_all_segments`: 3 segments → translate() called once per hop, not 3 times
- `test_translate_segments_batch_string_uses_xml_markers`: the string passed to translate() contains `<1>`, `<2>`, `<3>` markers
- `test_translate_segments_parses_translated_markers`: when translate() returns `<1>foo</1><2>bar</2>`, output texts are ["foo", "bar"]
- `test_translate_segments_mismatch_raises`: when translate() returns only `<1>foo</1>` for a 2-segment input, RuntimeError("Batched translation marker mismatch: expected 2 segments, got 1") is raised
- `test_translate_segments_pivot_batches_each_hop`: 2-hop route (en→de→es) → translate() called once for en→de hop and once for de→es hop (2 total calls, not 2×segment_count calls)

**Engine dispatch tests:**
- `test_translate_segments_engine_deepl_missing_key`: monkeypatch load_settings() to return AppSettings(deepl_api_key=""), call translate_segments(segs, "en", "es", engine="deepl") → RuntimeError("DeepL API key not configured. Set deepl_api_key in settings.")
- `test_translate_segments_engine_libretranslate_missing_url`: monkeypatch load_settings() to return AppSettings(libretranslate_url=""), call translate_segments(segs, "en", "es", engine="libretranslate") → RuntimeError("LibreTranslate URL not configured. Set libretranslate_url in settings.")
- `test_translate_segments_engine_deepl_calls_translator`: monkeypatch deepl.Translator, call translate_segments(segs, "en", "es", engine="deepl") → deepl.Translator(api_key).translate_text(texts, target_lang="ES") called with all segment texts as list
- `test_translate_segments_engine_libretranslate_calls_post`: monkeypatch requests.post to return mock response with {"translatedText": "hola"}, call translate_segments(segs, "en", "es", engine="libretranslate") → requests.post called with "http://localhost:5000/translate" and json payload
- `test_translate_segments_engine_unknown_raises_valueerror`: engine="unsupported_xyz" → ValueError("Unknown engine 'unsupported_xyz'. Choose argos, deepl, or libretranslate.")
  </behavior>

For the batching tests, configure `fake_translate.translate` as follows:
```python
# Return well-formed markers (mirrors what Argos hopefully returns)
def _make_translate_return(segments_texts):
    return "".join(f"<{i+1}>{t}</{i+1}>" for i, t in enumerate(segments_texts))

fake_translate.translate.side_effect = lambda batch, *args: _make_translate_return(
    # extract texts from the batch using the same regex the implementation uses
    [m.group(2) for m in re.finditer(r"<(\d+)>(.*?)</\1>", batch, re.DOTALL)]
)
```
