# Summary: Elapsed Time Counter Above Progress Bar

## What Was Built

Added a real-time elapsed time counter (`CTkLabel`) to `gensubtitles/gui/main.py` that appears above the progress bar during subtitle generation. The counter displays `HH:MM:SS` format, starts counting when generation begins, and shows the final elapsed time after completion (both success and error).

## Key Changes

- `import time` added (after `import threading`)
- `self._elapsed_timer` and `self._elapsed_start` instance attributes added
- `_elapsed_label` (CTkLabel) created at row 7, immediately hidden with `.grid_remove()`
- Progress bar shifted to row 8, stage label to row 9, Open Folder button to row 10
- `_tick_elapsed()` method added — counts up every second via `self.after(1000, ...)`
- `_on_generate` resets/shows the label and starts the tick loop
- `_finish_generate` cancels the timer but leaves the label visible with final time
