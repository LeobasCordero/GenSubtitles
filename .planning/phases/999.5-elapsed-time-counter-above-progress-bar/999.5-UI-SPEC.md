## Lifecycle Spec

| Event | Counter Behavior |
|-------|-----------------|
| App start / at rest | Hidden (`grid_remove`) — not visible |
| `_on_generate` called | Reset text to `"00:00:00"`, record `self._elapsed_start = time.monotonic()`, call `.grid()` to show, start 1-second tick loop |
| Every 1000ms tick | Compute elapsed = `int(time.monotonic() - self._elapsed_start)`, update label text, reschedule with `self.after(1000, self._tick_elapsed)`, guard with `self._closing` check |
| `_finish_generate` called | Cancel tick via `self.after_cancel(self._elapsed_timer)`, leave label **visible** with final time — do NOT call `.grid_remove()` |
| Clear button pressed | No effect on counter — label stays visible with last elapsed time |
| Next `_on_generate` called | Reset and restart (same as initial generate flow) |
| Window closing (`self._closing = True`) | Tick guard prevents further updates |
