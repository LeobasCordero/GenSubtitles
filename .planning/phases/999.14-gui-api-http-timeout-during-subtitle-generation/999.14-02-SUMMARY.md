## Summary

Replaced the blocking `_run_api_call` + polling `_poll_progress` with a three-step SSE-based `_run_sse_flow`. Added a Cancel button that appears only during active generation and calls `DELETE /subtitles/{job_id}` in a background thread.

## Key Files

### Modified
- `gensubtitles/gui/main.py` — Multiple targeted edits: added `_current_job_id`, added `_btn_cancel` widget at row 11, added `_on_cancel` handler, replaced `_run_api_call` with `_run_sse_flow`, removed `_poll_progress` and `_poll_in_flight`, updated `_on_generate` to show cancel button, updated `_finish_generate` to hide cancel button.

## Changes

| Item | Before | After |
|------|--------|-------|
| HTTP pattern | Single blocking POST with 3600s timeout | POST async → SSE stream → GET result |
| Progress | Polling `GET /progress` every second | SSE events applied via `self.after(0, ...)` |
| Cancel | Not available | Cancel button (row 11), hidden by default |
| Timeout | `timeout=3600` (HTTP read timeout) | `timeout=None` for SSE stream |
| Error labels | Raw exception string | `data.get("label")` from SSE event |
