# 999.2-01 Summary: Clear Button with Reactive Enable/Disable

## What Was Built

Added a "Clear" button to the GenSubtitles GUI that resets all 4 input fields in one click. Button is context-aware: disabled on launch, enables reactively when any field has content, and is disabled during pipeline execution.

## Key Files

### Modified
- `gensubtitles/gui/main.py` — Added `_btn_clear` (row 5), `_update_clear_state()`, `_on_clear()`, and `StringVar.trace_add` on all 4 input vars. Shifted progress bar → row 6, stage label → row 7, open folder → row 8.

## Decisions Made

- One shared `lambda *_: self._update_clear_state()` trace callback registered on all 4 vars (simpler than per-var callbacks)
- `state="disabled"` initial value set directly in CTkButton constructor (not a separate configure call)
- `_update_clear_state()` called in `_finish_generate` (not `_on_generate`) so re-evaluation happens after pipeline ends with fields still populated
