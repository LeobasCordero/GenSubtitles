## Spacing Scale

All padding and gap values MUST come from `gensubtitles/gui/styles.py` constants:

| Token | Constant | Value | Usage |
|-------|----------|-------|-------|
| xs | `SPACING_XS` | 4 px | Icon gaps, tight inner padding |
| sm | `SPACING_SM` | 8 px | Row gaps, compact element spacing |
| md | `SPACING_MD` | 12 px | Medium inner padding (widget interior) |
| lg | `SPACING_LG` | 16 px | Section inset / standard outer padding |
| xl | `SPACING_XL` | 24 px | Larger row gaps, before/after primary actions |
| xxl | `SPACING_XXL` | 32 px | Tab content horizontal inset |

Exceptions: none — no ad-hoc magic numbers.

Widget heights:
| Token | Constant | Value | Usage |
|-------|----------|-------|-------|
| Primary button height | `BTN_HEIGHT_PRIMARY` | 44 px | "Generar", "Extraer", "Transcribir", "Traducir", "Escribir", "Limpiar" |
| Browse button width | `BTN_WIDTH_BROWSE` | 80 px | "Explorar…" / "Guardar como…" side buttons |
| Progress bar height | `PROGRESS_BAR_HEIGHT` | 16 px | Tab 1 progress bar only |

## Typography

All font values MUST be resolved via `font(role)` from `gensubtitles/gui/theme.py`.

| Role | Function call | Size | Weight (Dark/Light) | Usage |
|------|---------------|------|---------------------|-------|
| Header | `font("header")` | 20 px | normal / bold | Section titles (e.g. "Configuración") |
| Subheader | `font("subheader")` | 16 px | normal / bold | Panel subtitles, label groups |
| Body bold | `font("body_bold")` | 14 px | normal / bold | Form labels, button text |
| Body | `font("body")` | 14 px | normal | General text, dropdown values |
| Mono | `font("mono")` | 12 px Courier | normal | Log textboxes (all tabs) |

Mode-aware weight: bold roles render as `weight="normal"` in Dark mode to avoid visual heaviness — this is handled by `font()` automatically; **do not override**.

### Tabs 3–6 — Step Tabs — Shared Structure

Each step tab follows the same visual pattern:

```
Tab frame (fill both)
└─ outer_frame (CTkFrame transparent, pack fill=both expand=True, padx=SPACING_XXL, pady=SPACING_LG)
   │  grid layout: columnconfigure(1, weight=1)
   │
   ├─ Row 0: Label(s("…_input_lbl")) col=0 sticky=w  +  Entry(_xxx_input_var) col=1 sticky=ew  +  btn "Explorar…" col=2
   ├─ [Row 1: Label(s("…_output_lbl")) col=0         +  Entry(_xxx_output_var) col=1 sticky=ew  +  btn "Guardar…" col=2]  — only if tab has output field
   ├─ [Rows: dropdown labels + CTkOptionMenu widgets — tab-specific]
   ├─ Spacer row: pady=SPACING_XL
   ├─ btn_row CTkFrame (transparent), pack
   │    ├─ btn "Acción"  (accent, BTN_HEIGHT_PRIMARY, width fills ~half)
   │    └─ btn "Limpiar" (secondary, BTN_HEIGHT_PRIMARY, width fills ~half)
   │    justify: side=LEFT with padx=SPACING_SM between
   └─ _xxx_log_textbox: CTkTextbox
        font=font("mono"), fg_color=p("input_bg"), state=disabled
        pack fill=both expand=True, pady=(SPACING_SM, 0)
```

Row padding: `pady=SPACING_SM` between form rows; `pady=(0, SPACING_XS)` for label rows.

## Interaction Contract

### Button states
| State | Condition | Visual |
|-------|-----------|--------|
| Disabled | Server not ready | `state="disabled"` (CTK default opacity) |
| Enabled | Server ready + required fields filled | `state="normal"` |
| Disabled (running) | Step currently executing | `state="disabled"` during API call |

Primary action buttons (Tabs 3-6): disabled until server ready (`_on_server_ready()` enables them, same pattern as `_btn_generate`).

### Pre-fill behavior (D-04)
Silent, automatic — no confirmation dialog. User can overwrite the pre-filled value.
Pre-fill triggers after successful API response, executed on main thread via `self.after(0, ...)`.

### Error display (D-03)
All errors displayed in the log textbox of the active/originating tab. No `messagebox` / popup dialogs for step errors.

### Log textbox behavior (all tabs)
- `state="disabled"` when idle (prevents user edits)
- `configure(state="normal")` → `insert("end", msg + "\n")` → `configure(state="disabled")` → `see("end")`
- Background color: `p("input_bg")` (theme-aware)
- Font: `font("mono")` (12 px Courier)
