Brief intro: Automatic video subtitle generation — offline, no API keys required. Transcribes and optionally translates video audio using Whisper and Argos Translate.

**## CLI Usage**

4 focused bash examples (per D-04, D-05):
1. Basic: `python main.py --input video.mp4`
   (Output auto-derived to video.srt per D-03 from Phase 7)
2. Custom output: `python main.py --input video.mp4 --output subtitles.srt`

Document all 6 flags: `--input`, `--output`, `--model`, `--target-lang`, `--source-lang`, `--device`

**## API Usage**

### Starting the server
```bash
python main.py serve
# or
uvicorn gensubtitles.api.main:app --host 0.0.0.0 --port 8000
```

### Endpoints

#### POST /subtitles
Curl example (D-04: bash + curl only):
```bash
curl -X POST http://localhost:8000/subtitles \
  -F "file=@video.mp4" \
  -F "target_lang=es" \
  --output subtitles.srt
```

Parameters: `model_size` (query), `target_lang` (query or form), `source_lang` (query or form)

#### GET /languages
```bash
curl http://localhost:8000/languages
```
Returns JSON: `{"pairs": [...]}`

Mention `/docs` for interactive Swagger UI.

**## Language Translation**

Document Argos Translate behavior per D-03:
- First use downloads language model pair (internet required, ~50-200MB)
- Models are cached locally (OS-appropriate cache directory)
- Subsequent runs are fully offline
- Use `--target-lang` with ISO 639-1 code (e.g., `es`, `fr`, `de`)

**## Troubleshooting**

Exactly 3 failure modes per D-07:

### FFmpeg not found
**Error:** `EnvironmentError: FFmpeg not found in PATH`
**Solution:** Install FFmpeg (see Installation section), verify with `ffmpeg -version`, restart terminal.

### Argos model download failure
**Error:** Network timeout or HTTP error during model download
**Solution:** Check internet connection, retry command. Models download once and are cached locally.

### Missing output directory
**Error:** `FileNotFoundError` or permission error writing SRT
**Solution:** Ensure output directory exists and is writable, or use `--output` to specify a valid path.
