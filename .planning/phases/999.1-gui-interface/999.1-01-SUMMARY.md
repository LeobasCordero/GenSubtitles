# 999.1-01 Summary: GUI Package Foundation

## What Was Built

Created the `gensubtitles/gui` Python package with a full CustomTkinter window skeleton and Uvicorn server auto-start/stop lifecycle.

## Key Files

### Created
- `gensubtitles/gui/__init__.py` — Package marker with docstring
- `gensubtitles/gui/main.py` — `GenSubtitlesApp(ctk.CTk)` class with full 6-row widget grid, progress bar, stage label, server thread lifecycle, and browse dialogs

### Modified
- `pyproject.toml` — Added `customtkinter>=5.2.0` and `requests>=2.31.0` to `[project.dependencies]`; added `gensubtitles-gui` script entry point
- `requirements.txt` — Added `customtkinter>=5.2.0` after existing requests line

## Decisions Made

- `ctk.set_appearance_mode("System")` / `ctk.set_default_color_theme("blue")` set at module level (safe for headless import, only affects display rendering)
- All `tkinter.filedialog` / `tkinter.messagebox` imports are deferred inside method bodies — module imports cleanly with no `DISPLAY` error
- Server thread is daemon so it exits automatically if the main thread exits unexpectedly
- `uvicorn.Server.should_exit = True` used for clean shutdown (uvicorn's documented flag)
- `OSError` on server start is caught and logged — handles the case where port 8000 is already in use
