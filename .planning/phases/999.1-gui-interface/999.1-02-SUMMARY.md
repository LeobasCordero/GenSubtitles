## Decisions Made

- `requests` import is deferred inside `_run_api_call` body (not module-level) — matches deferred-import pattern used across the codebase, avoids ImportError at lint time
- `device` and `model` forwarded as query params to `/subtitles` — matches API router's expected signature
- `platform.system()` used inside `_show_success` to open OS file manager cross-platform (Windows: `explorer`, macOS: `open`, Linux: `xdg-open`)
- `threading.Thread(daemon=True)` for API call thread — doesn't block window close
- `self.after(0, callback)` used to marshal API results back onto the Tk main thread safely
