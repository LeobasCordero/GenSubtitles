  <name>Task 2: Implement GenSubtitlesApp window with server thread and full UI layout</name>
  <files>gensubtitles/gui/main.py</files>
  <action>
    Create `gensubtitles/gui/main.py` implementing `GenSubtitlesApp(ctk.CTk)` and a `main()` entry function.

    **Server lifecycle (per D-06):**
    - Import `uvicorn`, `threading`, `signal`
    - `_start_server()`: creates a `uvicorn.Config("gensubtitles.api.main:app", host="127.0.0.1", port=8000, log_level="error")` server and runs it with `uvicorn.Server(config).run()` in a daemon thread. Store the `Server` instance on `self._server`.
    - `_stop_server()`: calls `self._server.should_exit = True` (uvicorn's clean-shutdown flag).
    - `on_closing()`: calls `_stop_server()` then `self.destroy()`. Bind to protocol `"WM_DELETE_WINDOW"`.
    - `__init__`: calls `_start_server()` at the end of setup.

    **UI layout (per D-01, D-02, D-03):**
    Use `ctk.CTk` with `appearance_mode="System"` and `color_theme="blue"`. Window title: `"GenSubtitles"`. Minimum size 540×440.

    Use a single `ctk.CTkFrame` as the main container with 20px padding. Grid layout within it.

    Row-by-row widgets (grid column 0=label, column 1=entry/widget, column 2=browse button):
    - **Row 0 — Input Video:** `ctk.CTkLabel("Input video:")`, `ctk.CTkEntry` (textvariable=`self._input_var`), `ctk.CTkButton("Browse…", command=self._browse_input)`
    - **Row 1 — Output SRT:** `ctk.CTkLabel("Output SRT:")`, `ctk.CTkEntry` (textvariable=`self._output_var`), `ctk.CTkButton("Save as…", command=self._browse_output)`
    - **Row 2 — Model:** `ctk.CTkLabel("Model size:")`, `ctk.CTkOptionMenu` with values `["tiny","base","small","medium","large-v1","large-v2","large-v3","turbo"]` default `"small"` (variable=`self._model_var`)
    - **Row 3 — Device:** `ctk.CTkLabel("Device:")`, `ctk.CTkOptionMenu` with values `["auto","cpu","cuda"]` default `"auto"` (variable=`self._device_var`)
    - **Row 4 — Source lang:** `ctk.CTkLabel("Source language:")`, `ctk.CTkEntry` hint `"auto-detect"` (textvariable=`self._source_lang_var`)
    - **Row 5 — Target lang:** `ctk.CTkLabel("Target language:")`, `ctk.CTkEntry` hint `"(none — no translation)"` (textvariable=`self._target_lang_var`)
    - **Row 6 — Generate button:** `ctk.CTkButton("Generate Subtitles", command=self._on_generate)` spanning columns 0–2, stored as `self._btn_generate`
    - **Row 7 — Progress bar:** `ctk.CTkProgressBar(mode="indeterminate")` spanning columns 0–2, stored as `self._progress_bar`. Initially hidden (`.grid_remove()`).
    - **Row 8 — Stage label:** `ctk.CTkLabel("")` spanning columns 0–2, stored as `self._stage_label`.

    **Browse callbacks (stubs — called from UI buttons):**
    - `_browse_input()`: `filedialog.askopenfilename(filetypes=[("Video files","*.mp4 *.mkv *.avi *.mov *.webm"),("All files","*.*")])` → set `self._input_var`
    - `_browse_output()`: `filedialog.asksaveasfilename(defaultextension=".srt", filetypes=[("SRT subtitles","*.srt")])` → set `self._output_var`

    **`_on_generate` stub** (implementation comes in Plan 02):


    **`main()` function:**
    ```python

        app.mainloop()
    ```

    **Error handling for server port conflict:** wrap `uvicorn.Server(config).run()` in a try/except; log any `OSError` at ERROR level (server may already be running).

    All `tkinter` imports (`filedialog`, `messagebox`) must be imported inside method bodies (lazy) to avoid display errors in headless environments.
