# Phase 999.1: GUI Interface - Context

**Gathered:** 2026-04-10
**Status:** Ready for planning (updated — D-11 through D-14 added)

<domain>
## Phase Boundary

Add a graphical user interface so non-technical users can generate subtitles without using the CLI or REST API. The GUI wraps the FastAPI REST API (phases 8–9 must be complete first) with a modern desktop application using CustomTkinter. Delivers a full-featured window mirroring all CLI flags, auto-launches the Uvicorn server on startup, and shuts it down on close.

</domain>

<decisions>
## Implementation Decisions

### Framework
- **D-01:** Use **CustomTkinter** as the GUI framework — modern-looking Tkinter wrapper, one pip dep (`customtkinter`), portable, no system installation required beyond Python.
- **D-02:** Desktop app (not a local web UI like Gradio/Streamlit) — runs as a native window.

### Feature Scope
- **D-03:** GUI exposes the 4 API-supported parameters as UI controls: input file, output path, source language, target language. Model size and device are not per-request configurable — they are fixed at server startup (lifespan model loading) and excluded from the GUI.
- **D-04:** Output path is **always required** — always show a Save dialog (no auto-derive default). This intentionally differs from CLI behavior to give GUI users explicit control.

### Integration
- **D-05:** GUI calls the **FastAPI REST API** (localhost) — not `run_pipeline()` directly, not CLI subprocess.
- **D-06:** GUI **auto-launches the Uvicorn server** in a background thread on window startup, and stops it when the window closes. User does not need to start the server manually.
- **D-07:** This phase **depends on Phases 8 and 9** being complete (FastAPI REST API Core + Extensions).

### Progress & Feedback
- **D-08:** Show a **CustomTkinter progress bar** + stage label (e.g., `[2/4] Transcribing…`) during generation.
- **D-09:** On success: display a success message and an **Open Folder** button that opens the output directory in the OS file explorer.
- **D-10:** On error: show a **modal error dialog** (messagebox popup) with the error message. No inline error label.

### Language Controls
- **D-11:** Both source and target language use **free-text entry fields** with placeholder text. Source language placeholder is **"auto-detect"**. Target language placeholder is **"(none — no translation)"**. Dropdowns populated from `GET /languages` deferred to a future phase.

### Server Readiness
- **D-12:** **Status indicator label only** — no gate on the Generate button. A status label shows `"⏳ Loading model…"` while Uvicorn is starting and `"✓ Ready"` once the server responds. Generate button is always enabled; connection errors fall through to the existing modal error dialog.

### Entry Points
- **D-13:** Both a **standalone script** (`gensubtitles-gui = "gensubtitles.gui.main:main"` in `pyproject.toml`) and a **CLI subcommand** (`gensubtitles gui`) are provided.

### Theme
- **D-14:** Default appearance mode is **`System`** (follows OS dark/light setting) set at import time via `ctk.set_appearance_mode("System")`. A runtime theme selector widget is deferred to a future phase.

### Agent's Discretion
- Exact layout and widget arrangement within the window (grid vs pack, column widths, padding).
- API endpoint URL and port — use Uvicorn default (8000) unless there's a conflict.
- Whether to disable the Generate button while processing to prevent double-submission.

</decisions>

<code_context>
## Existing Code Insights

### Reusable Assets
- `gensubtitles/api/main.py` — FastAPI app entry point; GUI will call its endpoints
- `gensubtitles/api/routers/subtitles.py` — Subtitle generation endpoint (to be implemented in Phase 8)
- `gensubtitles/core/pipeline.py` — `run_pipeline()` with 4-stage progress callback — awareness of stages informs progress bar design
- `gensubtitles/cli/main.py` — 6 Typer flags; GUI mirrors these exactly (minus auto-derive for output)

### Established Patterns
- Progress stages: `[1/4] Extracting audio`, `[2/4] Transcribing`, `[3/4] Translating`, `[4/4] Writing SRT` — use these as stage label text in GUI
- Errors are plain strings from pipeline/API — display as-is in error dialog

### Integration Points
- FastAPI server: will run on `http://localhost:8000` (Uvicorn default)
- GUI entry point: add `gensubtitles/gui/main.py` (new module) and expose via `main.py` or a new CLI subcommand
- `requirements.txt` / `pyproject.toml`: `customtkinter` dependency must be added

</code_context>

<specifics>
## Specific Ideas

- "Portable desktop app" — runs from the Python environment, no installer required
- GUI should feel modern relative to standard Tkinter (CustomTkinter dark/light theme support)
- Auto-launch + auto-teardown of Uvicorn means the user just opens the window and clicks Generate — no terminal interaction

</specifics>

<deferred>
## Deferred Ideas

- **Portable app packaging** — Bundle as a standalone executable (e.g., PyInstaller) so users don't need Python installed. Intentionally out of scope for this phase; noted for a future phase.

</deferred>
