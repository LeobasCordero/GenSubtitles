## Framework Choice

**Q:** What type of GUI should this be?
- Options: Local web UI (Gradio/Streamlit), Tkinter, PyQt6/PySide6, CustomTkinter
- **User input:** "desktop app, portable, not installation needed"

**Q:** For a portable desktop app — which approach fits best?
- Options: Tkinter (built-in), CustomTkinter (one pip install), PyInstaller bundle
- **Selected:** CustomTkinter — one pip install, modern appearance

---

## Feature Scope

**Q:** What should the MVP GUI include?
- Options: Minimal (file picker + generate only), Intermediate (key settings), Full (all CLI flags)
- **Selected:** Full — mirror every CLI flag in the UI

**Q:** For the output path field, should it auto-derive (like CLI) or always require the user to choose?
- Options: Auto-derive if blank (matches CLI), Always show Save dialog
- **Selected:** Output path required — always show a Save dialog

---

## Integration Approach

**Q:** How should the GUI call the subtitle generation pipeline?
- Options: Import pipeline directly, Call CLI via subprocess, Call FastAPI REST API
- **Selected:** Call FastAPI REST API — requires phases 8–10 to be complete first

**Q:** The FastAPI server needs to be running. How should the GUI handle this?
- Options: GUI detects + shows Start Server button, User's responsibility, Auto-launch on open/close
- **Selected:** Auto-launch — GUI starts/stops the API server internally on open/close

---

## Progress & Feedback

**Q:** How should progress during subtitle generation be shown in the GUI?
- Options: Progress bar + stage label, Log panel, Spinner only
- **Selected:** Progress bar + stage label — e.g., [2/4] Transcribing… with a visual bar

**Q:** What happens when generation completes successfully?
- Options: Success message + open folder button, SRT preview, Status label only
- **User input:** "success message, open folder available through a button"
- **Captured:** Success message + Open Folder button

**Q:** How should errors during generation be displayed?
- Options: Inline error label (red), Error dialog (messagebox), Agent's discretion
- **Selected:** Error dialog — messagebox popup
