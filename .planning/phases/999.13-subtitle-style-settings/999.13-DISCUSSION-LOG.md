## Color Input Method

| Option | Description | Selected |
|--------|-------------|----------|
| Native OS color picker | `tkinter.colorchooser.askcolor()` — OS dialog, no new dependency, visual | ✓ |
| Hex text entry | Type `#FFFFFF` directly — compact, no visual feedback | |
| Named color dropdown | "White", "Yellow", "Black"... — safe, limited choices | |

**User's choice:** Native OS color picker
**Notes:** Swatch button approach — small colored button shows current value, click opens OS dialog pre-filled with current color.

## Font Family Input

| Option | Description | Selected |
|--------|-------------|----------|
| Dropdown of common subtitle fonts | Curated list, avoids invalid font names | ✓ |
| Free text entry | Full flexibility, typos silently fail | |
| Agent's Discretion | Agent picks sensible default | |

**User's choice:** Dropdown, specifically smart TV-compatible fonts only
**Notes:** Font list: Arial, Helvetica, Verdana, Trebuchet MS, Tahoma — chosen for smart TV compatibility (Samsung, LG, Sony).
