## Summary

Created `gensubtitles/api/routers/steps.py` with 4 path-based step endpoints for GUI use, mounted in `api/main.py`, with 8 tests in `tests/test_api_steps.py`.

## What Was Built

### gensubtitles/api/routers/steps.py
- `POST /steps/extract` — `{video_path, work_dir}` → `{status: done, output_path}`
- `POST /steps/transcribe` — `{work_dir}` → `{status: done, output_path}` (uses pre-loaded transcriber via Depends)
- `POST /steps/translate` — `{work_dir, target_lang}` → `{status: done, output_path}`
- `POST /steps/write` — `{work_dir, output_path}` → `{status: done, output_path}`
- Pydantic request/response models (ExtractRequest, TranscribeRequest, TranslateRequest, WriteRequest, StepResponse)
- FileNotFoundError mapped to HTTP 404; other errors to HTTP 422

### gensubtitles/api/main.py
- Added `from gensubtitles.api.routers.steps import router as steps_router`
- Added `app.include_router(steps_router)`

### tests/test_api_steps.py
- 8 tests covering all endpoints, success paths, error paths (404 for missing files), and transcriber injection
