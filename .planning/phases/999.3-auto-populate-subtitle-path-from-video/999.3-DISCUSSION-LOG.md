## Area 3: Path Derivation

**Question:** What is the exact output path formula, and are there edge cases?

**Analysis presented:**
- Formula: `str(Path(video_path).with_suffix(".srt"))`
- Edge case 1 (no extension): `Path("movie").with_suffix(".srt")` → `movie.srt` — no issue
- Edge case 2 (no directory): Not possible in browse-only mode; `askopenfilename` always returns absolute paths

**Conclusion:** Formula is unambiguous, no special handling needed.
