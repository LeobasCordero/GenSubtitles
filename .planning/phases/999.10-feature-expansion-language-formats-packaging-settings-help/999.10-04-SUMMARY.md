# Summary: 999.10-04 — Translate Subtitles Tab

## What Was Built

1. **CTkTabview wrapper** — existing Generate form wrapped in "Generate Subtitles" tab; "Translate Subtitles" second tab added

2. **`_build_translate_tab()` method** — 9-row form in Translate tab:
   - Input subtitle file browse (SRT/SSA/ASS)
   - Output path browse
   - Source language CTkOptionMenu
   - Target language CTkOptionMenu
   - "Convert only" checkbox (disables lang dropdowns)
   - Translate / Convert button
   - Progress bar + elapsed timer + stage label

3. **Action handlers**:
   - `_on_translate()` — validates inputs, kicks off background thread
   - `_run_translate()` — calls `translate_file()` or `convert_srt_to_ssa/convert_ssa_to_srt` based on `convert_only`
   - `_finish_translate()` — re-enables controls, shows success/error messagebox
   - `_tick_tl_elapsed()` — per-second elapsed timer for translate tab

4. **`_populate_language_dropdowns()` updated** — also populates Translate tab source/target dropdowns from installed pairs
