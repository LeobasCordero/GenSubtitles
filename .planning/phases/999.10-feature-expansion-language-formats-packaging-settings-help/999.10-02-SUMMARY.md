# Summary: 999.10-02 ‚Äî translate_file() + CLI subcommands

## What Was Built

1. **`translate_file()` in `gensubtitles/core/translator.py`**
   - Reads SRT or SSA files via pysubs2 (auto-detects format)
   - Defaults `source_lang` to `"en"` with a logged warning if not provided
   - Raises `FileNotFoundError` on missing input, `ValueError` on same-lang pair
   - Outputs file preserving input extension (pysubs2 infers format from extension)

2. **New CLI subcommands in `gensubtitles/cli/main.py`**
   - `gensubtitles translate --input file.srt --target-lang es` ‚Äî offline subtitle translation
   - `gensubtitles convert input.ssa output.srt` ‚Äî format conversion (SRT‚ÜîSSA)

3. **`--format srt|ssa` flag on `generate` command**
   - Default: `srt`. When `ssa`: converts post-generation, removes temp `.srt`
