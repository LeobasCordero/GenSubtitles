    Create `gensubtitles/core/settings.py` implementing settings persistence using platformdirs.

    ```python
    """
    gensubtitles.core.settings
    ~~~~~~~~~~~~~~~~~~~~~~~~~~
    Application settings persistence.
    Settings are stored as JSON in the OS user config directory.
    """
    from __future__ import annotations
    import json
    import logging
    from dataclasses import asdict, dataclass, field
    from pathlib import Path
    logger = logging.getLogger(__name__)
    ```

    **`AppSettings` dataclass:**
    ```python
    @dataclass
    class AppSettings:
        appearance_mode: str = "System"   # "Light" | "Dark" | "System"
        ui_language: str = "en"           # "en" | "es"
        default_output_dir: str = ""      # absolute path or "" (use same dir as input)
        default_source_lang: str = ""     # ISO 639-1 code or "" (Whisper auto-detect)
    ```

    **`DEFAULT_SETTINGS = AppSettings()`** (module constant)

    **`settings_path() -> Path`:**
    - `from platformdirs import user_config_dir` inside the function
    - Return `Path(user_config_dir("GenSubtitles")) / "settings.json"`

    **`load_settings() -> AppSettings`:**
    - Call `settings_path()`; if file does not exist, return `AppSettings()` (defaults)
    - `json.loads(path.read_text(encoding="utf-8"))`
    - Construct `AppSettings(**{k: v for k, v in data.items() if k in AppSettings.__dataclass_fields__})`
    - Catch any exception (json.JSONDecodeError, KeyError, etc.), log a warning, return `AppSettings()`

    **`save_settings(settings: AppSettings) -> None`:**
    - `path = settings_path()`
    - `path.parent.mkdir(parents=True, exist_ok=True)`
    - `path.write_text(json.dumps(asdict(settings), indent=2), encoding="utf-8")`

    No external state; pure functions + dataclass. No mutable globals.
