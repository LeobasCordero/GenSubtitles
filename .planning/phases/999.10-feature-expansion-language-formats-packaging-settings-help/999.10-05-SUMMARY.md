# Summary: 999.10-05 — Menu Bar + Settings Panel

## What Was Built

1. **`_build_menu_bar()`** — creates and attaches a `tkinter.Menu` to the CTk root:
   - `Settings > Preferences…` → opens settings panel
   - `Help > Tutorial`, `Help > Available Languages`, `Help > About GenSubtitles` → stubs

2. **`_build_settings_panel()`** — creates an in-window settings CTkFrame (not packed initially):
   - Appearance Mode dropdown (System / Light / Dark)
   - UI Language dropdown (English / Spanish)
   - Default output directory entry
   - Save button (persists + applies immediately)
   - Back button (dismisses without saving)

3. **`_show_settings()` / `_hide_settings()`** — swaps tabview ↔ settings panel using `pack_forget` / `pack`

4. **`_save_settings()`** — creates `AppSettings` from panel values, calls `save_settings()`, applies `ctk.set_appearance_mode()` immediately

5. **`_apply_startup_settings()`** — called in `__init__` after `_start_server()`; loads persisted settings, applies appearance mode

6. **Help stubs** — `_show_tutorial`, `_show_language_pairs`, `_show_about` (all `pass`; Plan 06 implements)
