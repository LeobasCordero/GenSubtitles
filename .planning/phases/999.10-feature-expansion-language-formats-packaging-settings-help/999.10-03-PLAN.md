<objective>
Upgrade the GUI's language selection from static hardcoded values to a dynamic system driven by the installed Argos Translate models queried from `GET /languages`.

Changes:
1. **Dynamic language loading** — `_load_language_pairs()` fetches `GET /languages` after server starts, populates both source and target dropdowns (D-01)
2. **Re-enable source language dropdown** — shows available source languages (hidden in Phase 999.9); uses `CTkOptionMenu` matching target lang pattern (D-01)
3. **Cascading target filtering** — selecting a source lang filters target dropdown to only valid destinations for that source (D-02)
4. **Output format dropdown** — SRT / SSA selector in the generate form; changes output extension on browse and _run_api_call (D-08)

Purpose: Users can only select language pairs that are actually installed. UI stays consistent with real capabilities.
Output: Updated gui/main.py with dynamic dropdowns, source lang re-enabled, output format dropdown.
</objective>

    def _on_source_lang_change(self, selection: str) -> None:
        """Filter target language dropdown to valid destinations for selected source."""
        if selection == "Auto-detect" or not self._language_pairs:
            targets = list({_CODE_TO_LABEL.get(p["to"], p["to"]) for p in self._language_pairs})
            targets = sorted(targets) or ["No target"]
        else:
            src_code = _label_to_code(selection)
            targets = [
                _CODE_TO_LABEL.get(p["to"], p["to"])
                for p in self._language_pairs if p["from"] == src_code

            targets = sorted(set(targets)) or ["No target"]

        self._target_lang_var.set(targets[0])

    **Step 5 — Add _populate_language_dropdowns:**
    ```python
    def _populate_language_dropdowns(self) -> None:
        """Called after server is ready. Queries GET /languages and updates dropdowns."""
        import requests as req
        try:
            resp = req.get(f"{_BASE_URL}/languages", timeout=5)
            resp.raise_for_status()
            self._language_pairs = resp.json().get("pairs", [])
        except Exception as exc:
            logger.warning("Could not load language pairs from API: %s", exc)
            self._language_pairs = []

        if not self._language_pairs:
            return  # leave defaults
        sources = sorted({_CODE_TO_LABEL.get(p["from"], p["from"]) for p in self._language_pairs})
        sources = ["Auto-detect"] + sources
        self._option_source_lang.configure(values=sources)
        self._source_lang_var.set("Auto-detect")
        self._on_source_lang_change("Auto-detect")  # populate target
    ```
